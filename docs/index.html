Bridge Playwright Tests to Gherkin Syntax

A powerful VS Code extension that converts your Playwright test files to readable Gherkin syntax with live preview capabilities. Transform your test scenarios into clear, BDD-style documentation instantly.

Input: Playwright Test
test.describe('User Authentication', () => {
  test.beforeEach(async ({ page }) => {
    await test.step('the user is on the login page', async () => {
      await page.goto('https://example.com/login');
    });
  });

  test('should login with valid credentials', async ({ page }) => {
    await test.step('enter username', async () => {
      await page.fill('#username', 'testuser');
    });

    await test.step('enter password', async () => {
      await page.fill('#password', 'password123');
    });

    await test.step('click login button', async () => {
      await page.click('#login-button');
    });

    await test.step('verify successful login', async () => {
      await expect(page.locator('.dashboard')).toBeVisible();
    });
  });
});

Installation

1

Download Extension

Download the latest .vsix file from the releases page.

2

Install in VS Code

Install using VS Code command palette or terminal command:

code --install-extension playwright-bdd-bridge-0.1.2.vsix
3

Start Using

Open a TypeScript test file and use the preview button or right-click menu to start converting.

Usage

Command Palette

Press Ctrl+Shift+P (or Cmd+Shift+P on Mac) and type "Playwright: Open BDD Preview"

Context Menu

Right-click on any TypeScript test file in the explorer and select "Playwright: Open BDD Preview"

Preview Button

Click the preview button (üìÑ) in the editor title bar when viewing a TypeScript test file
